Detailed Description

Write CloudTurbine-formatted data. While not a full one-to-one implementation of the Java CTwriter API, this C# version provides basic functionality for storing different types of data. Data can optionally be packed and/or ZIP'ed at the block level. Timestamps can either be represented as milliseconds or seconds since epoch. All timestamps are auto-generated (i.e., the user cannot supply timestamps).
